ENABLE ACCESSIBILITY

Allow WriterPro to type into other apps with one quick setup.

Before you start

WriterPro needs Accessibility + Input Monitoring so it can detect your text box and type into other apps.

Enable both permissions once, then you’re done.

Step‑by‑step (macOS)

1

Open System Settings

Open the Apple menu, then choose System Settings.

Screenshot placeholder
Drag & drop Apple menu screenshot
2

Go to Privacy & Security → Accessibility

Privacy & Security → Accessibility in the left sidebar.

Screenshot placeholder
Drag & drop Accessibility settings screenshot
3

Unlock changes

Click the lock icon, then confirm with Touch ID or your password.

Screenshot placeholder
Drag & drop unlock dialog screenshot
4

Enable WriterPro

Find WriterPro in the list and toggle it on.

Screenshot placeholder
Drag & drop toggle-on screenshot
5

Return to WriterPro

Return to WriterPro, click “I’ve Enabled Both,” and continue.

Screenshot placeholder
Drag & drop WriterPro confirmation screenshot
6

Go to Privacy & Security → Input Monitoring

In System Settings, open Input Monitoring.

Screenshot placeholder
Drag & drop Input Monitoring screen
7

Enable WriterPro (Input Monitoring)

Toggle WriterPro on so the app can detect your clicks.

Screenshot placeholder
Drag & drop toggle-on screenshot
8

Confirm Enabled and Restart App

When Accessibility and Input Monitoring both show Enabled, click "Restart App" to finish setup.

Screenshot placeholder
Drag & drop enabled + restart screen

Not seeing WriterPro?

  • Open WriterPro once, then return to Accessibility or Input Monitoring.
  • Use the “+” button to add WriterPro manually if needed.
  • Restart WriterPro if the toggle doesn’t appear.

Quick check

  • Open WriterPro and start a typing session.
  • If the app prompts again, return to Accessibility and confirm the toggle is on.
