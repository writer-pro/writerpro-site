ENABLE ACCESSIBILITY

Allow WriterPro to type into other apps with one quick setup.

Before you start

Accessibility access lets WriterPro control typing in your target app. It’s required for typing sessions.

You only need to enable this once.

Step‑by‑step (macOS)

1

Open System Settings

Apple menu → System Settings.

Screenshot placeholder
2

Go to Privacy & Security → Accessibility

Select Accessibility from the left sidebar.

Screenshot placeholder
3

Unlock changes

Click the lock icon and confirm with Touch ID or password.

Screenshot placeholder
4

Enable WriterPro

Find WriterPro in the list and toggle it on.

Screenshot placeholder
5

Return to WriterPro

Back in the app, click “I’ve Enabled Access”.

Screenshot placeholder

Not seeing WriterPro?

  • Open WriterPro once, then return to Accessibility.
  • Use the “+” button to add WriterPro manually if needed.
  • Restart WriterPro if the toggle doesn’t appear.

Quick check

  • Open WriterPro and start a typing session.
  • If the app prompts again, return to Accessibility and confirm the toggle is on.
