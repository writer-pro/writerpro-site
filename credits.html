Earn credits (guide)

Follow these steps to earn credits. We’ll add screenshots and videos next.

After you finish an offer, you’ll receive a code. You must redeem that code on the Billing page to add credits.

Step‑by‑step

1

Sign in

Make sure you’re logged in with the account you want to credit.

Screenshot placeholder
2

Open the earn credits flow

Start the earn-credits flow and complete the offer. When you finish, you’ll see a redemption code.

Screenshot placeholder
3

Copy your redemption code

Copy the code exactly as shown. Each code can only be used once.

Screenshot placeholder
4

Redeem on Billing

Go to Billing → “Redeem a code,” paste your code, and submit. Your balance updates immediately.

Screenshot placeholder
5

Check your balance

If the balance doesn’t refresh, click “Refresh” on the Billing page.

Screenshot placeholder

Help

  • Codes are not automatic—redeem them on the Billing page.
  • Each code can be used once and must match exactly.
  • If credits don’t appear, refresh the billing page.
  • Most credits post within a few minutes.
  • Contact support if a credit is missing after 24 hours.
